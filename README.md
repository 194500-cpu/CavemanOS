# Caveman OS
People tell teenagers to "hop off" their screens and do things they used to do when they were young, like go explore outside.
How about we combine them?
Caveman OS has the main aim of providing a clean, easy to use Web OS. 

Just like caves have drawings, every button on this page has the chance to contain a secret/easter egg, telling you more about the creator of it, or the mystery of the universe?

Interested? Go check it out!
