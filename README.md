# Caveman OS
A game about finding easter eggs, disguised as a WebOS.


People tell teenagers to "hop off" their screens and do things they used to do when they were young, like go explore outside.
How about we combine them?
Caveman OS has the main aim of providing a clean, easy to use Web OS. 

Just like caves have drawings, every button on this page has the chance to contain a secret/easter egg, telling you more about the creator of it, or the mystery of the universe?

Features:
-Lots of easter eggs
-A Campfire messaging app
-Supabase database to store messages in Campfire messaging app


Now, I would like to tell you a bit about what I learned.

Throughout this build, I learned about:
-JavaScript events and features
-linking a project to a database
-making my own icons and images using VectorPea

It was a great journey of learning and fun.
